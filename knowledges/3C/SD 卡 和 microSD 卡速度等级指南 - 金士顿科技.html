SD 卡 和 microSD 卡速度等级指南 - 金士顿科技

SD 协会提出了一种实现闪存卡速度级别标准化的方法。这被称作速度等级，指的是 SD 卡和 microSD 卡的持续性写入绝对最低速度。速度等级包含三种类型：

  • Speed Class
  • UHS Speed Class
  • Video Speed Class

紧靠或环绕着圆形“C”符号、“U”符号或“V”符号的数字代表闪存卡的速度等级。制造商利用这些速度等级符号，在闪存卡上指示速度等级类型和分级。它们旨在帮助消费者为他们的硬件设备选择速度合适的闪存卡。

Speed Class​

第一个速度等级被称作原始速度等级，以符号“C”表示。Speed Class 包含四个级别：

  • C2 (Class 2)：最低写入速度为 2MB/秒
  • C4 (Class 4)：最低写入速度为 4MB/秒
  • C6 (Class 6)：最低写入速度为 6MB/秒
  • C10 (Class 10)：最低写入速度为 10MB/秒

C2 是最慢的速度等级，而 C10 是行业标准。由于更新的硬件设备需要具备更高速度和更强功能的更新速度等级，今天几乎没人使用 Class 2–6。C10 通常是当今多数硬件设备的最低速度等级要求，是市场上发货量最大的闪存卡。

UHS Speed Class​

更高一级的速度等级是 UHS (Ultra-High Speed) Speed Class，以符号“U”表示。UHS Speed Class 包含两个级别：

  • U1 (UHS Speed Class 1)：最低写入速度为 10MB/秒
  • U3 (UHS Speed Class 3)：最低写入速度为 30MB/秒

今天，UHS Speed Class 比 Speed Class 更常用，许多高端相机要求为高分辨率视频录制等功能使用至少 U3 级别的闪存卡。UHS Speed Class 主要是指视频录制的最低持续性写入性能，旨在支持需要更快写入速度的 4K 视频录制设备。一般来说，支持 4K 录制的相机通常需要至少 U3 级别的 SD 卡。

U1 和 U3 闪存卡比 Speed Class 闪存卡更先进，因为它们使用两个 UHS 总线接口中的一个：

  • UHS-I：高达 104MB/秒的理论最高传输速度
  • UHS-II：高达 312MB/秒的理论最高传输速度

U1 和 U3 闪存卡都可利用 UHS-I 总线接口，但不兼容 UHS-II 总线接口。

这些 UHS 总线接口指示的是理论最高读取和写入速度，与速度等级的持续性写入速度不同。UHS 总线接口以罗马数字“I”或“II”符号标示在卡的正面。总线速度是指接口本身的理论数据传输速率，而 U3 级别的 SD 卡拥有自己的持续性写入速度（30MB/秒）。例如，UHS-I U3 级别的卡保证 30MB/秒的写入速度，但如果与支持 UHS-I 总线接口的设备搭配使用，则潜在读写速度高达 104MB/秒。

UHS-II 兼容卡的潜在读写速度高达 312MB/秒。UHS 总线接口向下兼容，因此您可以在支持 UHS-I 的设备中使用 UHS-II 卡，但无法发挥 UHS-II 的速度优势，因为此卡默认采用较低的 UHS-I 规格。卡和总线接口必须完全兼容，才能体验到速度优势。

Video Speed Class​

最新的 Video Speed Class 旨在用于支持更高的视频分辨率和录制功能，例如视频流、360 度拍摄、VR 内容，以及 4K 或 8K 分辨率视频。它们通常以符号“V”表示。Video Speed Class 包含五个级别：

  • V6 (Video Speed Class 6)：最低写入速度为 6MB/秒
  • V10 (Video Speed Class 10)：最低写入速度为 10MB/秒
  • V30 (Video Speed Class 30)：最低写入速度为 30MB/秒
  • V60 (Video Speed Class 60)：最低写入速度为 60MB/秒
  • V90 (Video Speed Class 90)：最低写入速度为 90MB/秒

Video Speed Class 的独特性在于，它可以利用 UHS-I 和 UHS-II 总线接口。V6 至 V90 速度等级闪存卡可以使用 UHS-II 总线接口，但 UHS-I 总线接口仅支持 V6 至 V30 速度等级闪存卡。

Video Speed Class 提供最快的可用速度，非常适合无人机和 360 度摄像机的超高分辨率视频、高品质视频和多文件录制。它支持无人机、360 度摄像机、运动摄像机和 VR 摄像机中高达 8K 的高清格式。

您可以为自己的硬件设备选择相同或更高的速度等级，找到最适合您的卡。例如，如果设备要求使用 Class 4 闪存卡，您可以使用 Speed Class 4、6 或 10。如果设备要求使用 UHS Speed Class 1 卡，您可以使用 UHS Speed Class 1 或 3。这同样适用于 Video Speed Class。注意，您可以使用超过设备规定速度等级的卡，但无法体验到更高速度等级的优势，因为设备仅支持更低的速度等级。
